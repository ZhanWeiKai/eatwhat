# 今天吃什么 - 前端项目

这是一个朋友/情侣之间讨论"今天吃什么"的社交应用前端原型。

## 📱 项目简介

根据产品需求文档（PRD）开发的前端原型，实现了以下核心功能：

- ✅ 欢迎页导航
- ✅ 讨论空间（实时查看好友推荐）
- ✅ 上传菜品（图片+名称+推荐理由）
- ✅ 邀请好友（二维码+链接分享）
- ✅ 我的推送（历史记录管理）
- ✅ 用户绑定（微信登录）
- ✅ **权限控制**：只有自己推送的菜品可以删除

## 🎨 UI 风格

- **主色调**：橙色 #FF6600（食欲感、活力）
- **设计风格**：简洁现代、卡片式布局
- **移动端优先**：APP风格、触摸友好
- **动画效果**：页面切换、按钮反馈、弹窗动画

## 🚀 快速开始

### 方法 1：直接打开（推荐）

1. 双击打开 `index.html` 文件
2. 或者右键 → 使用浏览器打开

### 方法 2：使用本地服务器（推荐用于开发）

如果你安装了 Python：

```bash
# Python 3
cd test-frontend
python -m http.server 8000

# 然后访问: http://localhost:8000
```

如果你安装了 Node.js：

```bash
# 安装 http-server
npm install -g http-server

# 启动服务器
cd test-frontend
http-server -p 8000

# 然后访问: http://localhost:8000
```

### 方法 3：使用 VS Code Live Server

1. 安装 VS Code
2. 安装 "Live Server" 扩展
3. 右键 `index.html` → "Open with Live Server"

## 📱 页面说明

### 1. 欢迎页 (index.html)
- 品牌展示区
- 4个主要功能入口
- 橙色渐变背景

### 2. 讨论空间页 (space.html)
- **顶部**：空间信息栏（成员头像墙）
- **中部**：菜品推送列表
- **底部**：上传按钮（悬浮）
- **权限控制**：只有自己的推送显示删除按钮

### 3. 上传菜品页 (upload.html)
- 图片上传（支持拍照和相册）
- 菜品名称（必填，最多20字）
- 推荐理由（选填，最多100字）
- 表单验证和成功提示

### 4. 邀请好友弹窗
- 二维码展示
- 分享邀请链接功能
- 成员列表显示

### 5. 我的推送页 (my-pushes.html)
- 查看历史推送记录
- 所有推送都可以删除
- 空状态提示

### 6. 用户绑定页 (profile.html)
- 微信快捷登录
- 用户信息展示
- 数据统计（推送数、空间数、点赞数）

## 🎯 核心功能演示

### 权限控制（重要）
```
自己的推送 → 显示删除按钮 ✅
他人的推送 → 不显示删除按钮 ❌
```

### 删除流程
1. 点击删除按钮
2. 弹出确认对话框
3. 确认后执行删除
4. 所有成员的列表同步更新（模拟）

### 上传菜品流程
1. 选择图片 → 预览
2. 输入名称和描述
3. 点击"推送"
4. 显示成功提示
5. 自动返回空间页

## 🎨 设计规范

### 颜色
- 主色：#FF6600（橙色）

- 卡片间距：12px
- 元素间距：8px
- 按钮间距：15px
- 圆角：8-20px

## 🐛 已知问题

1. **图片占位符**：使用了 placeholder 图片，实际开发需要替换为真实图片
2. **扫码功能**：目前只显示 toast 提示，需要集成真实扫码 SDK
